## Move

- `cd -` : go to previous directory

## Type
- `C-u` : delete all line 
- `C-w` : delete word

## Search

- ctrl + r `type keyword to search history command`
- `!nvim <tab>`
- `!! <Enter>`



## Kill process running port

```sh
lsof -i:3000
kill <PID>
```

## 目錄大小

```sh
du -sh <path>
```

## Library
