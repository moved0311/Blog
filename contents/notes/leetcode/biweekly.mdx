Q2. Maximize the Total Height of Unique Towers

```py
def maximumTotalSum(self, maximumHeight: List[int]) -> int:
	res = []
	picked = set()

	for n in maximumHeight:
		tmp = n
		while tmp in picked:
			tmp -= 1
		if tmp <= 0:
			return -1
		picked.add(tmp)
		res.append(tmp)

	return sum(res)
```
