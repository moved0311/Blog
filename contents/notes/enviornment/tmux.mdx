| shortcut | description       |
| -------- | ----------------- |
| C-a ?    | search command    |
| C-a ,    | rename window     |
| C-a c    | create new window |
| C-a n    | go to next window |
| C-a p    | go to prev window |
| C-a z    | zoom pane         |
