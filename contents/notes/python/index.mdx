## Sort
```py
list.sort()
list.sort(reverse=True)
```

```py
t = [('a', 5), ('e', 1),('c', 3), ('b', 2)]
t.sort(key=lambda c: c[1])
for i in t:
	print(i)
# ('e', 1)
# ('b', 2)
# ('c', 3)
# ('a', 5)
```

## Counter
```py
cnt = Counter()
s = "aaabbcc"

for c in s:
	cnt[c] += 1

# Counter({'a': 3, 'b': 2, 'c': 2})

for k, v in cnt.items():
	print(k, v)

# a 3
# b 2
# c 2
```

```py
s = "aaaabbb"
s.count('a') # 4
```
