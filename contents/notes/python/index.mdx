## Counter
```py
cnt = Counter()
s = "aaabbcc"

for c in s:
	cnt[c] += 1

# Counter({'a': 3, 'b': 2, 'c': 2})

for k, v in cnt.items():
	print(k, v)

# a 3
# b 2
# c 2
```
